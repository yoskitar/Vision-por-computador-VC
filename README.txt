Herramienta para recuperación de instancias mediante modelos de bolsa de palabras.
Para reproducir la experimentación realizada ejecutando el archivo fuente main.py, deberemos incluir en la estructura de carpetas vacia las imágenes
junto con las palabras para cada una siguiendo el mismo formato que se muestra en el enlace de referencia y que podemos descargar de éste.
En el documento podemos encontrar más información del proyecto, así como ejemplo de ejecución y modo de uso.
